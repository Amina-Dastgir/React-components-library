# React Component Library

## Project Description
This project demonstrates a reusable React component library containing:
- Button Component
- Card Component
- Modal Component

These components can be reused in different React applications.

## Installation Instructions

1. Clone repository

git clone <repository-link>

2. Navigate to project

cd react-component-library

3. Install dependencies

npm install

4. Start development server

npm run dev

## Usage Guidelines

### Button

<Button text="Click Me" />

### Card

<Card

/>
